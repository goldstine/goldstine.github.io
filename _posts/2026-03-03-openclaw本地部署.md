openclaw config set channels.feishu.appId "XXXX"
# 设置飞书App Secret
openclaw config set channels.feishu.appSecret "XXXX"
# 启用飞书渠道
openclaw config set channels.feishu.enabled true

1.打开飞书开放平台（https://open.feishu.cn/app），用飞书账号登录。                
2.点击创建企业自建应用，填写任意应用名称，选择应用类型为机器人，点击创建。                 
3.进入应用详情页，点击凭证与基础信息，复制App ID和App Secret。             
4.在飞书应用详情页，点击权限管理，开启以下核心权限（缺一不可）：              
▪im:message（消息发送/接收）               
▪im:chat（群聊访问）           
▪im:message.p2p_msg:readonly（单聊消息读取）           
▪application:bot.menu:write（机器人菜单编辑）          
▪im:message:send_as_bot（以机器人身份发送消息）          
▪contact:contact.base:readonly（获取通讯录基本信息）         
▪contact:user.id:readonly（通过手机号或邮箱获取用户ID）

四、常用命令说明              
安装完成后，日常操作全靠命令行搞定，不用记复杂指令，掌握下面这些常用命令，就能轻松管理OpenClaw（所有系统通用，复制粘贴即可执行，重点命令标红提示），每一条都配了通俗解读，小白也能一眼看懂。           
         
（1）核心服务管理命令（最常用，必记！）         
这类命令用于控制OpenClaw的核心服务（Gateway），启动、停止、查看状态都靠它们，日常用得最多，记牢这几条就够了：                   

•启动服务：openclaw daemon start，解读：最基础的启动命令，安装完成后首次启动、服务停止后重启，都用这条，执行后AI助手开始运行，飞书机器人即可正常响应。           
•停止服务：openclaw daemon stop，解读：需要修改配置、安装新Skill后，必须先停止服务，再重启才能生效，执行后飞书机器人会停止响应，服务关闭。          
•重启服务：openclaw daemon restart，解读：修改配置、安装新插件后，快速重启服务的捷径，相当于停止+启动，不用输两条命令。             
•查看OpenClaw版本：openclaw --version，解读：查看当前安装的OpenClaw版本，判断是否需要更新。         
•查看帮助文档：openclaw --help解读：忘记命令用法时，执行这条，会显示所有命令的详细说明，相当于随身手册。        
（2）大模型（基模）相关命令     
用于查看、修改当前使用的大模型（基模），比如想从Qwen换成Claude，或者查看已配置的模型，用这些命令：            

•查看当前使用的基模：openclaw models，解读：执行后会显示当前配置的大模型（比如qwen-3-plus），以及模型的基础信息（API地址、调用额度等）。              
•修改基模（切换模型）：openclaw models set 模型名称，解读：核心命令！比如想从Qwen切换到Claude，执行openclaw models set claude-3-opus，切换后重启服务即可生效；常用模型名称：qwen-3-plus（通义千问）、claude-3-opus（Claude）、gpt-4-turbo（OpenAI）。            
•查看支持的所有模型：openclaw models list，解读：想知道OpenClaw支持哪些大模型，执行这条就能看到完整列表，包括模型名称、支持状态、适用场景。              
（3）Skills（技能）相关命令          
Skills是AI的工具箱，安装、卸载、查看、更新Skill，全靠这些命令，和我们后面实操股票助手密切相关：          

•安装新Skill：npx clawhub install Skill名称，解读：最常用！比如安装股票推荐Skill，执行npx clawhub install 1password，安装完成后重启服务即可使用。           
•卸载不需要的Skill：npx clawhub uninstall Skill名称，解读：Skill装多了占内存，不用的可以卸载，比如卸载代码生成Skill，执行npx clawhub uninstall 1password。            
•查看已安装的所有Skill：openclaw skills list，解读：查看自己装了哪些Skill，以及每个Skill的版本、运行状态（正常/异常）。            
（4）Hooks（钩子）相关命令        
Hooks是AI的自动触发开关，添加、删除、查看钩子，实现AI自动响应，比如收到关键词就执行Skill，用这些命令：                  
     
•查看已添加的所有Hook：openclaw hooks list，解读：查看自己添加的所有钩子，包括触发条件、执行动作、状态。      
•启用/禁用Hook：openclaw hooks enable/disable Hook名称解读：不想删除Hook，但暂时不用，执行这条命令禁用，需要时再执行一次启用，比如openclaw hooks disable boot-md。              
五、常见问题的解决方案（持续更新）        
（1）首次与飞书对话报错      
OpenClaw: access not configured.        
Your Feishu user id: ********              
Pairing code: XXXXXXXX       
Ask the bot owner to approve with:                 
openclaw pairing approve feishu XXXXXXXX               
其中，XXXXXXXX是飞书的配对码，执行如下命令即可修复：         

# 执行授权命令，替换成你的配对码         
openclaw pairing approve feishu XXXXXXXX         

# 检查当前授权状态          
openclaw pairing list         

# 测试OpenClaw基础功能      
openclaw feishu send --chat-id <你的飞书群ID> --text "测试消息"
