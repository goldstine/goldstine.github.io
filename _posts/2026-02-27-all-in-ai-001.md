三、基础使用流程（从文本到成片）
▪导入文本：上传小说 / 剧本，或直接粘贴内容。
▪AI 解析：自动提取角色、场景，生成标准化剧本。
▪生成分镜：一键生成分镜列表，可编辑镜头、画面、台词。
▪配置模型：选择大模型与绘图模型，设定画风与分辨率。
▪批量生成：自动生成分镜画面，合成配音与视频。
▪导出成片：下载 MP4 视频，支持二次剪辑。
▪视频教程: https://www.bilibili.com/video/BV1na6wB6Ea2﻿




四、GitHub 部署方法
1. 环境准备
•Node.js 16+、Yarn、Git、PM2（可选，用于守护进程）
•服务器：2 核 4G 以上，建议 Linux 系统
2. 克隆代码
git地址:

 https://github.com/HBAI-Ltd/Toonflow-app 

# GitHub克隆
git clone https://github.com/HBAI-Ltd/Toonflow-app.git
cd Toonflow-app

# 国内Gitee克隆（速度更快）
git clone https://gitee.com/HBAI-Ltd/Toonflow-app.git
cd Toonflow-app
3. 安装依赖与构建
yarn install
yarn build
4. 启动服务
•开发模式：yarn dev
•生产模式（PM2 守护）：
pm2 start yarn --name "toonflow" -- start
pm2 save
pm2 startup
5. 访问
浏览器打开：http://localhost:3000（本地）或服务器IP:3000

五、大模型配置方法
Toonflow 为模型编排层，需手动配置大模型与生成模型：

1.进入后台模型配置页面
2.大语言模型（剧本 / 分镜）：支持通义千问、文心一言、GPT、DeepSeek、本地 LLaMA 等，填写 API Key / 地址与参数。
3.图像 / 视频模型（分镜画面）：支持即梦、Stable Diffusion、Midjourney API、万相、本地模型，配置接口与风格参数。
4.保存配置，测试连通后即可使用。
